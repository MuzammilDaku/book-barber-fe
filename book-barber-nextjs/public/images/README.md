# BookMyBarber Images Directory

This directory should contain the following images for the BookMyBarber website:

## Required Images

1. **Logo**
   - `logo.png` - The BookMyBarber logo

2. **Hero Section**
   - `hero-image.jpg` - Main image for the homepage hero section

3. **Barber Images**
   - `barber1.jpg` - Image for Royal Cuts Barbershop
   - `barber2.jpg` - Image for Classic Men's Parlor
   - `barber3.jpg` - Image for Modern Cuts Studio

4. **Authentication Pages**
   - `login-image.jpg` - Background image for login page
   - `register-image.jpg` - Background image for registration page

5. **Testimonials**
   - `testimonial1.jpg` - Image for first testimonial
   - `testimonial2.jpg` - Image for second testimonial

## Image Guidelines

- Use professional, high-quality images
- Optimize images for web (compress to reduce file size)
- Recommended dimensions:
  - Logo: 200x200px
  - Hero image: 1200x800px
  - Barber images: 600x400px
  - Authentication images: 800x1200px
  - Testimonial images: 200x200px (preferably circular crop)

## Image Sources

You can use free stock images from websites like:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)

Make sure to use images that are licensed for commercial use.
